\section{第1課 \furigana{李}{り}さんは\furigana{中}{ちゅう}\furigana{国}{こく}\furigana{人}{じん}です}

\subsection{基本课文}

\begin{enumerate}
    \item 李さんは中国人です。
    \item 森さんは学生ではありません。
    \item 林さんは日本人です。
    \item 李さんは日中企画の社員です。
\end{enumerate}


\begin{itemize}
    \item 甲：私は李です。小野さんですか。
    \item 乙：はい、そうです。小野です。
\end{itemize}

\begin{itemize}
    \item 甲：森さんは学生ですか。
    \item 乙：いいえ、学生ではありません。会社員です。
\end{itemize}

\begin{itemize}
    \item 甲：吉田さんですか。
    \item 乙：いいえ、ちがいます。森さんです。
\end{itemize}

\begin{itemize}
    \item 甲：李さんは日中企画の社員ですか。
    \item 乙：はい、そうです。
\end{itemize}

\subsection{应用课文}

李：日中企画の小野さんですか。

小野：はい、小野です。李秀麗さんですか。

李：はい、李秀麗です。初めまして。どうぞ　よろしく　お願いします。

小野：はじめましで。小野です。

森：李さん、こんにちは。

\vspace{1em}

李：吉田さんですか。

森：いいえ、私は　吉田では　ありません。森です。

李：あっ、どうも　すみません。

森：いいえ、どうぞ　よろしく。

李：こちらこそ、よろしく　お願いします。
