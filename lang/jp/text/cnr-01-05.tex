\section{第5課 森さんは７時に起きます}


\subsection{动词三分类}

一类动词：う段 $\rightarrow$ い段 + ます

\begin{enumerate}
    \item 结尾是う段非る的动词
    \item あ段+る、う段+る、お段+る
    \item 特例：切る、帰る、走る
\end{enumerate}

二类动词：る $\rightarrow$ ます

\begin{enumerate}
    \item い段+る
    \item え段+る
\end{enumerate}

三类动词

\begin{enumerate}
    \item \furigana{来}{く}る $\rightarrow$ \furigana{来}{き}ます
    \item する $\rightarrow$ します
\end{enumerate}


\subsection{基本課文}

\begin{enumerate}
    \item 今、四時です。
    \item 森さんは、七時に　起きます。
    \item 森さんは、先週　休みました。
    \item 私は　昨日　働きませんでした。
\end{enumerate}


\begin{itemize}
    \item 毎日、何時に　寝ますか。
    \item 十一時
\end{itemize}


\subsection{语法}

\begin{itemize}
    \item 〜さす、〜ません、〜ました、〜ませんでした。
    \item 名[时间]に　动词。
    \item 
\end{itemize}
